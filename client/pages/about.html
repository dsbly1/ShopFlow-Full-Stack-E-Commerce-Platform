What's Built In

A full platform, not a prototype

🔒

JWT Authentication

Secure login, email verification via Resend API, and role-based access for buyers, sellers, and admins.

🗄

Normalized Database

7-table PostgreSQL schema with indexes, a place_order() stored procedure, and a top products view.

🏪

Seller Dashboard

Full seller onboarding with 4-step registration, product management, payout tracking, and 1099 tax tools.

📦

Order Management

Cart, checkout, and inventory decrement handled atomically in a single transaction. Full order history included.

⭐

Reviews & Ratings

Verified buyer reviews with aggregate ratings surfaced directly in the product listing query via SQL JOIN.

🚀

Deployed on Vercel

Frontend and backend unified on Vercel — serverless, fast, and always available with zero cold-start delays.
